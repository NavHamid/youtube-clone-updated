# 🎬 YouTube Clone

A clean, responsive **YouTube homepage clone** built with **HTML, CSS, and JavaScript**.
The base version was inspired by SuperSimpleDev tutorials and then improved with additional UI behavior and styling.

## ✨ Overview

This project recreates the core YouTube homepage experience, including:

- a fixed header with search and action controls,
- a sidebar navigation layout,
- a responsive video grid,
- and category navigation interactions.

## 🚀 Features

- 📌 Fixed header with search bar and action icons
- 📂 Sidebar navigation layout
- 🎞️ Responsive video grid
- 🧭 Category bar interactions
- 🎨 Hover effects and modern UI styling

## 🛠️ Tech Stack

- **HTML5** for page structure
- **CSS3** for styling and responsiveness
- **Vanilla JavaScript** for interactivity

## ▶️ Run Locally

1. Clone or download this repository.
2. Open the project folder.
3. Open `youtube.html` in your browser.
