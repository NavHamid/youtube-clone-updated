# 🎬 YouTube Clone

A clean and responsive **YouTube homepage clone** built using **HTML, CSS, and JavaScript**, inspired by tutorials from SuperSimpleDev and enhanced with custom features and improvements.

---

## ✨ Overview

This project recreates the core layout and UI experience of YouTube’s homepage, including a responsive video grid, interactive sidebar, and a modern category navigation system.

While the foundation was inspired by learning resources, the project has been **extended and customized with original improvements** in UI behavior, structure, and interactivity.

---

## 🚀 Features

* 📌 Fixed header with search bar and action icons
* 📂 Collapsible and expandable sidebar navigation
* 🎞️ Responsive video grid layout
* 🧭 Scrollable category bar with smooth arrow controls
* 🌙 Dark mode toggle with dynamic UI updates
* 🎨 Interactive hover effects and modern UI styling

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Layout, responsiveness, and styling
* **JavaScript (Vanilla)** – Interactivity and UI logic

---

## 📁 Project Structure

```
📦 YouTube Clone
 ┣ 📄 youtube.html        # Main file (HTML + JS)
 ┣ 📂 styles/             # CSS files
 ┣ 📂 icons/              # UI icons
 ┣ 📂 thumbnails/         # Video thumbnails
 ┗ 📂 channel-picture/    # Channel images
```

---

## ▶️ Run Locally

1. Download or clone the repository
2. Open the project folder
3. Open `youtube.html` in your browser
